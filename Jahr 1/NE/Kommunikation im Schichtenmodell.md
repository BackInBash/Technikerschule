# Kommunikation im Schichtenmodell

## OSI Modell

| Layer | Funktion | Protokolle |
|--|--|--|
| L7 | Application | HTTP, FTP, SNMP |
| L6 | Presentation |
| L5 | Session |
| L4 | Transport | TCP, UDP, GRE, ICMP|
| L3 | Network | IPv4 / IPv6|
| L2 | Data-Link | 
| L1 | Physical |
